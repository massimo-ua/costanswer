Operating Controls

COSTANSWER uses 2 types of software operating controls – data entry fields and buttons.

Data entry fields are designed to be filled in with text and numbers and buttons are designed to save or update the entered data. Some fields have drop-down list options.

There are several simple rules to follow while using data entry fields and buttons:

You can select any item by clicking on it. The selected item changes color to green.

When you are done entering the data, press the “Save” button.

You can change a saved item by clicking on it. After the changes are done, press the “Update” button.

To create one more item simply press the “Update” button and you will be redirected to the empty form for a new item. After entering the data required, please press the “Add” button.
