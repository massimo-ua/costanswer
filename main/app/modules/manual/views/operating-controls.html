Operating controls

COSTANSWER uses 2 types of software operating controls – input fields and buttons.

Input fields are designed to be filled in with texts and values and buttons are designed to save or update the inputted data. Some input fields have dropping list options.

There are several simple rules to follow while using input fields and buttons:

1. You can select any element by clicking on it. Selected element changes color to green.

2. When you are done with text or value input, please, press button “Add” or “Save”:

3. You can introduce changes to saved element simply by clicking on it and, after changes are done, press button “Update” to be redirected to the empty input form of the new element:

4. To create one more element simply press button “Update” and you will be redirected to the empty input form of the new element. After entering the data required, please, press button “Add”:

5. You can delete any element simply by pressing “X” on the button:

6. The buttons “Save” and “Add” remain inactive until the moment you put some values in the input forms. If you do not have ready value for some costing element or you need to skip it you can put “0” value to move forward:

7. Always follow the green colored buttons from the top to the bottom to navigate your current element selection:
