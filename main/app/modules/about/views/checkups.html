Checkups

Every costing project report you create using COSTANSWER online software is protected by 2 logic self-testing checkups:

  • Cost of goods manufactured = Cost of all batches
  • Full cost of fixed manufacturing overhead = Total fixed manufacturing overhead allocated to all goods

Every product report also has 2 logic checkups:

  • Variable cost unit + Fixed cost unit = Cost of goods manufactured (COGM) per unit
  • Variable cost batch + Fixed cost batch = Cost of goods manufactured (COGM) per batch

WARNING: If you see ERROR instead of OK in your report’s checkup section, DO NOT use this report, but inform COSTANSWER as soon as possible.
