Checkups

Every costing project report you create using COSTANSWER online software is protected by 2 logic self-testing checkups:

  1. Cost of goods manufactured = Cost of all batches.
  2. Full cost of fixed manufacturing overhead = Total fixed manufacturing overhead allocated to all goods.

Every product report also has 2 logic checkups:

  1. Variable cost unit + Fixed cost unit = Cost of goods manufactured (COGM) per unit.
  2. Variable cost batch + Fixed cost batch = Cost of goods manufactured (COGM) per batch.

WARNING: In case if you see an ERROR instead of OK in your report’s checkup section, please, DO NOT take this report into a business consideration, but inform COSTANSWER about the mistake as soon as possible.
